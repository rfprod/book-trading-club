User Stories
User can view books available via Google Books API. Books can be added to user collection from full catalogue or randomly by typing a name in My Books section. Books claimed to be owned by registered users can be requested by other registered users. User can enter with GitHub credentials or create an account providing a user name and a password. Registered users can edit own email, full name, city, and state.
